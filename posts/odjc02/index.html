【教程•科普•杂谈】 对于个人资源分享者，如何选择合适分享姿势？

前言

众所周知，现在的环境肯定是会越来越严苛的，毒盘举报爆链严重，资源佬们苦于补档渐渐耗光精力弃坑，其结果就是催生了一批举报党和麦片哥
今年事又特别多，选择合适的网盘分享还是很有必要的

分享者选择网盘时需要考虑到的几个条件

•时效长，尽量避免补档，不然压力很大很累人的，最好还可以本地同步
•低成本，能白嫖最好
•容量大，5g10g的我觉得不行
•速度适中，适用于国内网络
•稳定后台较硬，不至于一声不吭的跑路
就我来说，资源分享也就图一乐，又不是我要下管伸手党的感受干嘛，太过迁就伸手党是没有好处的

说说目前的几个主流网盘
•度盘
让人又爱又恨的存在。和谐就不说了年年都有，但近期大范围封号搞得人人自危——许多资源佬的号说封就封，svip也不无法幸免。（现在还有灰产卖度盘小号，才一两块钱建议入几个）
并不是说度盘就不用了，只是作为主力盘实在不合适。个人倾向利用秒传链接做备份盘。
简单解释一下度盘炸档的原因和使用秒传链接的意义
由于在线解压和举报党的存在，导致文件MD5值被封禁，从而变成敏感资源无法生成链接分享，一旦资源变得敏感，只能只能够重新压制资源，然后重新上传分享。
秒传链接能无视资源是否敏感，直接将文件存进百度云，这就不需要分享者去排查哪里有问题了，不可能一下子全炸，被和谐部分伸手党自觉用其他分流补完

•微云
优点是下载（文件转发好友）不限速 但不充钱量实在太少（默认10g），而且听说最近也开始大规模和谐了，早晚和度盘一样，pass

•115
对普通用户不太友好，成本较高，但和谐力度应该没上面两个大

•谷歌
被墙就卡住了一大部分人，不过容量大可作备用分流
虽然利用GoIndex+CloudFlare反代可以直连，但有时速度实在感人，可作应急用

•MEGA
和google一样，被墙，虽然有免翻的下载器但仅限PC，而且免费的量也不够（默认50g）

•微软
onedrive版本较乱，但条件基本符合，个人较为推荐，接下来的教程和科普主要也是关于onedrive的

Onedrive几个常见版本的区别

Onedrive产品有两个，一个是Onedrive，归Onedrive团队开发，一个是Onedrive for business，归sharepoint团队开发。
Onedrive就是个人帐号类似outlook所使用的，最大支持单文件100GB，并且可以不限制文件大小复制文件，在Office365个人版和家庭版都是使用的这个。
Onedrive for business就是企业帐号，教育帐号工作账号都是这种，一般后缀为on.microsoft.com，Office365企业级订阅都是用的这个，最大只支持上传15GB文件
Onedrive for business世纪互联 也就是国内版，服务器在上海或北京

接下来主要介绍Office 365白嫖的有什么区别，正规付费/学校/公司提供等不在讨论之列
先上一张神图（水印别在意）

1.Microsoft Office365 A1
•无Office365桌面版.
•无Azure Directory API操作权限.
•有管理员存在,才能在第三方应用挂载.
•有OneDrive for Busniess空间(1T~5T).
•正常情况下,许可证数量无限制(unlimited).
Microsoft Office365 A1 学生邮箱申请的A1子账户倒是有可能是永久的，很多高校都有校友邮箱，而手头有账户的人拥有的都是真正的教育邮箱(这远非上面那种违规添加的子账号所用的认证邮箱可比)，几乎没听说有哪所高校花钱购买批量授权而微软默认给A1(只要教育邮箱验证就能过)，证明它们是微软信任的教育机构，然而功能烂(不支持桌面版Office登陆)。
缺点：第二、三点，无Azure Directory API操作权限，很多情况下都是无管理员的，这就意味着你无法使用Oneindex、Pyone、Cuteone、Oonelist等程序，一般情况下国内大学教育邮箱申请的基本上都是A1版本。如果有管理员，那么你用的是否放心不说，还得看管理员心情，所以说管理员这东西真的是把双刃剑。
除此之外最大的缺陷就是没有桌面版，这就相当于你的电脑里面安装的office还是得自己想办法。
优点：很多情况下没有管理员，这就意味着你存放在onedrive的文件相对安全，不用担心数据泄露。
同时，因为用的人少，国内教育邮箱资源相对少，所以稳定性较高。
同时听说一般微软在封号的时候，早些时候一般都是只封停管理员，现在在审查教育机构资质及学生资质的时候有大概率直接干掉全体帐号，所以，你的教育邮箱的好坏直接决定了你的账户是否安全。这些老帐号安全性较高，同时正规的大学邮箱安全也较高。但是注意了某某职业学院、某某艺术学校之类的就要注意了。些管理员被封的A1P子账户只能说暂时安全，因为管理员被封的原因很大程度上就是违规添加教育版A1P子账号(用的都是像类似@gr这样批量生产的不正规域名，淘宝卖家普遍给大家用的用于一次性验证的教育邮箱多半也是这种，因为这个世界上没有那么多的正规教育邮箱可供他们这样挥霍)，指不定哪天就不行了，但至少用个四五年问题应该不大；这类账号的好处就是资料信息安全，onedrive可以随意放东西，除了微软基本没人能看。

2.Office365 A1P
•有Office365桌面版.
•无Azure Directory API操作权限.
•有管理员存在,才能在第三方应用挂载.
•有OneDrive for Busniess空间(1T~5T).
•正常情况下,许可证数量无限制(unlimited).
•这个就是A1的Plus版本（教育版），来源多来自国外教育邮箱批量购买授权。
和A1的最大区别就是多了个桌面版，当然这个多出来的东西可不是一般功能，当然只用onedrive都没差。另外稳定性和优缺点基本如上，正规学校基本没问题，野鸡大学随时会挂。

3.Office365 E3 MSDN
•有Office365桌面版.
•有Azure Directory API操作权限.
•有OneDrive for Busniess空间(1T~5T).
•正常情况下,许可证数量25个.
E3 MSDN版本倒是有那么一丝永久免费的可能性(一切要看微软2020年以后对待开发者的态度了，因为第一批开发者用户已经免费续了3年)，因为申请难度相对高一点，门槛相对高一点，所以在稳定性上要高。
同时子号不多，也是该版本稳定性高的原因之一。这个优点相对于A1P多了一个Azure Directory API操作权限，搞开发的人很有用，普通老百姓无所谓了。
目前市场上能看到卖的都是之前留下来的存货用一个少一个每年(或每月)可自动免费续期,无需额外操作!
官方的描述:”This subscription will automatically renewed all the time.”
现在这个全局管理成品号炒到一千多你敢信？

4.Office365 Enterprise E3 for Symphony
这个东西可以授权的数量可以达到几百万个，你觉得靠谱吗？
一般用于某宝奸商忽悠小白

5.Office365 E3 Trial
•无Office365桌面版.
•无Azure Directory API操作权限.
•有OneDrive for Busniess空间(1T~5T).
•正常情况下,许可证数量5个.
E3版本一年试用版本，到期给钱，自己就可以申请，功能上只有个onedrive和简单API权限。
申请地址：https://products.office.com/zh-cn/business/office-365-enterprise-e3-business-software

*6.Office365开发版E5 *
和E3差不多。官方免费申请，区别是E3固定一年试用期，而E5是三个月自动续订一次（玄学）
申请地址：https://developer.microsoft.com/zh-cn/microsoft-365/dev-program

部署网盘程序

准备工作
最好自备上网工具
heroku官方： heroku.com
注：Heroku是一个支持多种编程语言的云平台。提供免费的应用托管且没有被墙，免备案
OneManager：https://github.com/qkqpttgf/OneManager-php
为什么我选OneManager而不是别的网盘程序？
•易上手
•支持多盘挂载
•支持heroku，白嫖免服务器
•主题简洁

1.通过heroku官网注册账户并登陆（验证码需过墙才能显示）

建议用谷歌邮箱注册 没有？参考这一篇：https://www.yundianseo.com/create-google-account
2.点击OneManager的：github仓库地址进入后下拉找到部署到heroku然后单击按钮

3.点击后跳转到创建新应用。app名称随便填，生成后分配的二级域名就是你的应用名称 地区可选美国或者欧洲。点击Deploy app部署应用，完成后点击view跳转部署


剩下按步骤就可以了 这里提示一下：
如果是世纪互联版不要勾选自己申请应用ID与机密
